图片滤镜（Image Filter）

基于 CSS 的 filter 属性，Marp 可以对图片进行一些基于模糊、亮度、对比度等的操作，如：

![blur:15px](image.png)
![brightness:0.5](image.png)
![contract:150%](image.png)

背景图片

同时通过在 bg 后追加图片的格式属性，如 [bg fit] ，可以具体设置背景图片的缩放方式。其中 cover 表示充满页面， fit 表示拉伸以适应页面， auto 为不做缩放使用原图片比例。
